# 3B Şasi Modelleri

Bu dizin, uydu şasisine ait `.STL` ve `.STEP` dosyalarını içermektedir.

## Tasarım Notları

| Parça | Malzeme | Doluluk | Ağırlık (tahmini) |
|:---|:---|:---:|:---:|
| Ana Gövde | PLA+ | %40 Gyroid | ~85 g |
| Üst Kapak | PETG | %30 Grid | ~20 g |
| Fin Aparatı | PLA+ | %50 Line | ~15 g |
| Servo Braketi | PETG | %80 Solid | ~8 g |

## Baskı Parametreleri

```
Katman Yüksekliği : 0.2 mm
Duvar Sayısı      : 3
Dolgu Deseni      : Gyroid (titreşim sönümleme için)
Sıcaklık (PLA+)  : 210°C nozül / 60°C tabla
Sıcaklık (PETG)  : 235°C nozül / 75°C tabla
```
